Rucsok

  • id : BigDecimal
    The identifier of the Rucsok
  • title : String
    The title of the rucsok, crawled from the site meta tags/title tag.
  • link : String
    The source the rucsok got crawled from.
  • imageUrl : String
    The image URL if present
  • videoUrl : String
    The video URL if present
  • type : String
    The type of the rucsok. Valid values are IMAGE, EMBEDVIDEO, HTML5VIDEO
  • username : String
    The user's name who posted the rucsok
  • vote : BigDecimal
    The aggregated number of upvotes/downvotes
