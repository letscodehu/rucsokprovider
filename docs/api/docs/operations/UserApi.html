profileGet

The User Profile endpoint returns information about the Rucsokprovider user that has authorized with the application.

URL

http://localhost//profile

HTTP Method

GET

Response Type

Profile

Parameters

registerPost

The endpoint used to register a new user into the Rucsokprovider with the role of user.

URL

http://localhost//register

HTTP Method

POST

Response Type

UserRegistrationResponse

Parameters

  • Query: email String

    The e-mail address of the user to be registered

  • Query: username String

    The username wanted

  • Query: password String

    The password used to login

  • Query: passwordConfirmation String

    The password confirmation
